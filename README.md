# 프로젝트 피클

프로젝트 피클은 **로컬 디렉토리의 동영상 파일을 스캔하고, 재생하고, 평가하고, 정리하는 Windows 데스크톱 앱**입니다.

핵심 목표는 다음과 같습니다.

- 지정한 디렉토리를 스캔하여 동영상 파일 정보를 DB에 적재한다.
- 재생 중심의 UI에서 동영상을 빠르게 검토할 수 있다.
- 파일명 변경, 태그, 설명, 스냅샷, 썸네일 지정 등 관리 기능을 제공한다.
- 로컬 개인용 환경에서 빠르고 안정적으로 동작한다.

---

## 1. 제품 개요

프로젝트 피클은 단순한 동영상 플레이어가 아니라, **동영상 평가 및 정리 도구**를 목표로 한다.

예상 사용 시나리오:

- 여러 폴더에 흩어진 동영상 자료를 한 번에 스캔한다.
- DB에 적재된 목록을 정렬/필터/검색하면서 확인한다.
- 중앙 플레이어로 동영상을 재생하며 품질을 평가한다.
- 태그, 설명, 평점, 상태를 남긴다.
- 필요 시 이름을 바꾸거나 삭제 후보로 분류한다.
- 대표 썸네일과 스냅샷을 생성해 나중에 다시 찾기 쉽게 만든다.

---

## 2. 기술 스택

### 확정 스택
- **Framework**: Qt 6
- **UI**: Qt Quick / QML
- **Language**: C++20
- **Build**: CMake
- **Database**: SQLite (Qt SQL)
- **Media Playback**: Qt Multimedia
- **Primary Platform**: Windows 10/11 x64

### 보조 도구(예정)
- `ffprobe`: 동영상 상세 메타데이터 추출
- `ffmpeg`: 스냅샷/썸네일 생성

---

## 3. 핵심 요구사항

### 3.1 스캔 및 DB 적재
사용자가 지정한 루트 디렉토리를 재귀적으로 스캔하여 동영상 파일을 수집한다.

최소 저장 정보:
- 파일명
- 전체 경로
- 파일 크기
- 해시값
- 수정 시각
- 확장자

추가 저장 예정 정보:
- 길이
- 비트레이트
- 프레임레이트
- 해상도
- 비디오 코덱
- 오디오 코덱
- 설명(description)
- 태그
- 평점
- 검토 상태
- 대표 썸네일 경로
- 마지막 재생 위치
- 마지막 재생 시각

### 3.2 UI 구조
- **중앙**: 동영상 플레이어
- **하단**: 재생바 및 주요 액션
- **우측 상단**: 라이브러리 리스트
- **우측 하단**: 상세 정보 패널

### 3.3 라이브러리 리스트
- 정렬
- 필터
- 검색
- 썸네일 표시/비표시 토글
- 선택 항목 표시
- F2 이름 바꾸기

### 3.4 상세 정보 패널
- 용량
- 길이
- 비트레이트
- 해상도
- 코덱 정보
- 태그 관리
- 설명 관리
- 스냅샷 생성
- 대표 썸네일 지정

## 4. 추가 핵심 기능

프로젝트 피클은 단순 스캔/재생 앱을 넘어서, 아래 기능을 단계적으로 지원한다.

### 검토/평가 기능
- 검토 상태: `미검토 / 검토중 / 보관 / 삭제후보 / 즐겨찾기`
- 평점: 1~5점
- 마지막 시청 위치 저장
- 최근 본 파일 표시

### 정리 기능
- 파일명 변경
- 설명 편집
- 태그 편집
- 대표 썸네일 지정
- 스냅샷 저장

### 라이브러리 기능
- 검색
- 정렬
- 복합 필터
- 스마트 뷰(예: 미검토만, 즐겨찾기만)

### 확장 예정 기능
- 중복 파일 탐지
- 손상 파일 탐지
- 일괄 태그 적용
- 일괄 이름 변경

## 7. 개발 원칙

- QML은 UI만 담당한다.
- 비즈니스 로직은 C++에 둔다.
- DB 스키마 변경 시 migration을 추가한다.
- 항상 빌드 가능한 상태를 유지한다.
- 한 번에 하나의 작은 마일스톤만 구현한다.
- 초기에는 예쁜 UI보다 데이터 흐름과 안정성을 우선한다.
